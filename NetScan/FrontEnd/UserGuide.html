Network Security Scanning Commands

Scanning Methodology Overview

Network scanning is a critical process for identifying vulnerabilities, understanding network topology, and ensuring security. This guide provides comprehensive scanning techniques using NetScanner>_ tool.

⚠️ Important: Always obtain proper authorization before scanning networks. Unauthorized scanning can be illegal and unethical.

Nmap Scanning Techniques

1. Forward-confirmed Reverse DNS (FCrDNS)

nmap -sn -Pn --script fcrdns [hostname]

Performs a Forward-confirmed Reverse DNS lookup to detect DNS configuration anomalies.

2. Validates forward and reverse DNS mappings

[hostname] can be a domain name or IP address

Helps identify potential DNS spoofing or misconfiguration

3. Vulnerability Specific Scans - Log4Shell Vulnerability Check

nmap --script "http-vuln-log4shell" [hostname]

Scans for Log4Shell vulnerability (CVE-2021-44228).

4. Apache Path Traversal Vulnerability

nmap --script=http-vuln-cve-2021-41773 [hostname]

Checks for CVE-2021-41773 Path Traversal in Apache Web Server 2.4.49.

Nmap Scanning Techniques

5. Fast Network Scan

nmap -F [hostname]

Quick scan of most popular 100 ports.

6. Standard Port Scan

nmap [hostname]

Basic port scanning on a remote system.

7. Specific Service Scan

nmap -sV -p 21,22,25,80,110,143,443,445 [hostname]

Targeted scan of common service ports with version detection.

8. Comprehensive Scan

nmap -A [hostname]

Detailed system information gathering including OS detection.

9. Determining Firewall Rules

nmap -sA [hostname]

Unfiltered ports return reset, making their status unknown. Filtered ports don’t respond or send ICMP errors.

Advanced Nmap Scanning Techniques

Reconnaissance Scans

  • nmap -sP [hostname] - Ping Scan
  • nmap -sn -Pn --script fcrdns [hostname] - Reverse DNS Lookup
  • nmap --script http-headers [hostname] - HTTP Headers Scan
  • nmap --script banner [hostname] - Banner Grabbing

Vulnerability Scans

  • nmap --script "http-vuln*" [hostname] - HTTP Vulnerabilities
  • nmap --script=http-vuln-cve-2021-41773 [hostname] - Specific CVE Check
  • nmap -sV --script=http-malware-host [hostname] - Malware Detection

Advanced Scanning

  • nmap -O [hostname] - OS Detection
  • nmap -sA [hostname] - Firewall Rule Detection
  • nmap --top-ports [x] [hostname] - Top Ports Scan
  • nmap -p- [hostname] - Full Port Scanning

Protocol-Specific Scanning

TCP Scanning

nmap -sT [hostname]

Full TCP connect scan for TCP-based services.

Additional TCP scanning options:

  • nmap -sS [hostname] - Stealth SYN scan
  • nmap -p- [hostname] - Full TCP port scan

UDP Scanning

nmap -sU [hostname]

Comprehensive scanning of UDP-based services.

Additional UDP scanning options:

  • nmap -sU -p- [hostname] - Full UDP port scan
  • nmap -sU -sV [hostname] - UDP scan with service version detection

Additional Security Scanning Techniques

Network Discovery

  • nmap -sn [network/cidr] - Network ping sweep
  • nmap -PR [network/cidr] - ARP discovery on local network

Advanced Detection

  • nmap --traceroute [hostname] - Network path tracing
  • nmap -sV --version-intensity 5 [hostname] - Aggressive version detection

Evasion Techniques

  • nmap -f [hostname] - Fragmented packets
  • nmap --data-length 25 [hostname] - Custom packet data length
